Skills

11のカスタム Skills

.claude/commands/ に配置。/コマンド名 または自然文で自動発動

/build-and-review Analyzer→Builder→Reviewer の3段階パイプラインを順次実行
/file-summarizer ファイル読込 → 要約 → 改善提案の3段階処理（引数で文字数指定可）
/code-report Python を静的解析して品質メトリクスを Markdown レポートに出力
/char-counter 改行±・スペース±の4パターンで文字数を正確にカウント
/security-checks OWASP Top 10 準拠でコードの脆弱性を CRITICAL/HIGH/MEDIUM/LOW 分類
/git-commit-writer git diff を分析して Conventional Commits 形式のメッセージを自動生成
/readme-generator プロジェクト構成を解析して README.md を自動生成
/performance-guide 計算量・メモリ・I/O のボトルネックを特定して改善例を提示
/database-tips DBクエリ最適化・インデックス戦略・N+1問題などをアドバイス
/changelog-generator git log を解析して CHANGELOG.md を自動生成
/env-doctor 開発環境の設定・依存関係の問題を診断してレポート
