// 02 — SUBAGENTS

7つのカスタム SubAgents

.claude/agents/ に配置。@エージェント名で呼び出し、または Claude が自動判断して委譲

🔍@analyzerSTEP 1

Python を静的解析。構造・問題点を HIGH / MEDIUM / LOW で分類してレポート出力

🔨@builderSTEP 2

分析レポートをもとにコードを改善。型ヒント・docstring を自動追加し # IMPROVED: でマーク

✅@reviewerSTEP 3

改善コードの網羅性・安全性・一貫性を検証。APPROVED / NEEDS REVISION で最終判定

📋@code-reviewerAgent

コード品質レビューの専門エージェント。問題点・改善提案をリスト形式で報告

📝@doc-writerAgent

Python コードを解析して関数・クラスの仕様を README 風 Markdown に自動生成

🧪@test-runnerAgent

正常系・異常系・境界値の unittest を自動生成・実行してサマリーを報告

✍️@seo-blog-writerAgent

SEO 構成で記事を執筆し char-counter スキルで文字数を ±5% 以内に自動最適化

11のカスタム Skills
